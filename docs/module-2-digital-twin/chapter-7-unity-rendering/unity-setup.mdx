# 7.1: Unity 2022 LTS Installation and Setup

## Overview

Install Unity 2022 LTS (Long Term Support) and the ROS-TCP-Connector package to enable ROS 2 integration.

**Time Required**: 30-40 minutes
**Difficulty**: Beginner
**Disk Space**: ~20GB

## Why Unity 2022 LTS?

**LTS (Long Term Support)** versions receive:
- 2 years of bug fixes and patches
- Stable API (no breaking changes)
- Better package compatibility
- Recommended for robotics projects

:::tip Version Compatibility
This chapter uses **Unity 2022.3 LTS**. Other versions (2021 LTS, 2023 LTS) work but may have different UI layouts. Avoid non-LTS versions for production.
:::

## Prerequisites Checklist

Before installing, verify you have:

- [ ] **Operating System**:
  - Windows 10/11 (64-bit) OR
  - Ubuntu 22.04 LTS OR
  - macOS 12+ (Intel or Apple Silicon)
- [ ] **GPU**: DirectX 11/12 (Windows) or OpenGL 4.5+ (Linux/Mac)
- [ ] **RAM**: 8GB minimum, 16GB recommended
- [ ] **Disk Space**: 20GB free
- [ ] **Internet**: Active connection for downloads

## ROS-TCP-Connector Installation

### What is ROS-TCP-Connector?

The **Unity Robotics ROS-TCP-Connector** package enables:
- Bidirectional ROS 2 topic communication
- Service and action support
- Message serialization/deserialization
- Cross-platform networking (Windows â†” Linux)

### Step 6: Create Test Project

2. Select **"3D (URP)"** template
   - URP = Universal Render Pipeline (better graphics)
3. Project Settings:
   - **Name**: `HumanoidVisualization`

4. Click **"Create Project"**

**First Launch**: Takes 2-5 minutes to import assets and compile shaders

### Step 7: Install ROS-TCP-Connector Package

Unity will now be open. Follow these steps:

1. Go to **Window â†’ Package Manager**
2. Click **"+"** (top-left) â†’ **"Add package from git URL..."**
3. Enter:
   ```
   https://github.com/Unity-Technologies/ROS-TCP-Connector.git?path=/com.unity.robotics.ros-tcp-connector
   ```
4. Click **"Add"**

**Installation Progress**: Watch bottom-right status bar (30-60 seconds)

:::info Alternative Installation Method
If git URL fails (firewall/proxy):
1. Download [ZIP from GitHub](https://github.com/Unity-Technologies/ROS-TCP-Connector/archive/refs/heads/main.zip)
2. Extract `com.unity.robotics.ros-tcp-connector` folder
3. In Unity: **Window â†’ Package Manager â†’ + â†’ Add package from disk**
4. Select `package.json` inside extracted folder
:::

## Troubleshooting

### Unity Hub Won't Install Editor

**Problem**: Installation fails or hangs

**Solutions**:
1. Check internet connection
2. Free up disk space (need 10GB+ free)
3. Disable antivirus temporarily
4. Download offline installer:
   - Visit [Unity Download Archive](https://unity.com/releases/editor/archive)
   - Download Unity 2022.3 LTS "Unity Editor" installer
   - Run manually

### Package Manager "Git URL" Fails

**Problem**: Error adding ROS-TCP-Connector from git URL

**Solutions**:
1. Install Git if missing:
   ```bash
   # Windows: Download from git-scm.com
   # Linux: sudo apt install git
   # macOS: xcode-select --install
   ```
2. Use HTTPS instead of SSH URL (already shown above)
3. Use alternative installation (see Step 7 note)

### ROS Settings Menu Missing

**Problem**: "Robotics" menu doesn't appear

**Solutions**:
1. Verify package installed:
   - **Window â†’ Package Manager**
   - Search "ROS-TCP-Connector"
   - Should show "Up to date"
2. Restart Unity Editor
3. Reimport package:
   - Right-click package in Package Manager
   - **"Reimport"**

### GPU Not Detected (Linux)

**Problem**: Unity runs slowly or shows "OpenGL not supported"

**Solutions**:
1. Install proprietary GPU drivers:
   ```bash
   # NVIDIA
   sudo ubuntu-drivers autoinstall
   sudo reboot

   # AMD
   sudo apt install mesa-vulkan-drivers mesa-utils
   ```
2. Verify GPU:
   ```bash
   glxinfo | grep "OpenGL renderer"
   # Should NOT show "llvmpipe"
   ```

### License Activation Fails

**Problem**: Can't activate Unity Personal license

**Solutions**:
1. Check internet connection
2. Sign out and sign back in:
   - Unity Hub â†’ Sign Out
   - Sign In again
3. Clear Unity Hub cache:
   ```bash
   # Windows: Delete C:\Users\USERNAME\AppData\Roaming\UnityHub
   # Linux: rm -rf ~/.config/UnityHub
   # macOS: rm -rf ~/Library/Application\ Support/UnityHub
   ```
   Then restart Unity Hub
