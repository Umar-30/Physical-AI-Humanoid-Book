## Key Concepts Preview

:::info What is Gazebo?
**Gazebo** is an open-source robotics simulator that provides:
- **Accurate Physics**: Realistic dynamics, collisions, and forces
- **Sensor Simulation**: Cameras, lidar, IMU, force-torque sensors
- **3D Visualization**: Real-time rendering of robots and environments
- **ROS 2 Integration**: Native support for ROS 2 topics and services
:::

### Gazebo Harmonic vs Classic

This chapter uses **Gazebo Harmonic** (the modern version), NOT Gazebo Classic:

| Feature | Gazebo Classic | Gazebo Harmonic |

## Common Questions

<details>
<summary><strong>Q: Can I use Gazebo Classic instead?</strong></summary>

While possible, we **strongly recommend Gazebo Harmonic** because:
- Better ROS 2 integration
- Superior performance for humanoid robots
- Active development and support
- All examples in this book use Harmonic

</details>

<details>
<summary><strong>Q: Will this work on Windows?</strong></summary>

**Partial support**. Recommended approach:
1. Install WSL2 (Windows Subsystem for Linux)
2. Install Ubuntu 22.04 in WSL2
3. Follow Linux instructions
4. Use VcXsrv or X410 for GUI display

Native Windows support for Gazebo is limited.

</details>

<details>
<summary><strong>Q: My computer doesn't have a dedicated GPU. Will it work?</strong></summary>

Yes, but with limitations:
- Gazebo will run in **headless mode** (no GUI)
- Physics simulation still works
- Use Unity (Chapter 7) for visualization instead
- Performance may be lower (50-100 Hz vs 100-500 Hz)

</details>

<details>
<summary><strong>Q: How long does installation take?</strong></summary>

Typical installation time:
- Fast internet: 10-15 minutes
- Slow internet: 30-45 minutes
- Total package size: ~2GB

</details>
