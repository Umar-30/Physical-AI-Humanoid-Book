# Chapter 1.4: Humanoid Modeling with URDF

## Focus: URDF syntax, joint/link definitions, kinematics basics
## Learning objectives: Create a simple humanoid URDF model

This chapter will introduce the Unified Robot Description Format (URDF) for modeling humanoids. We will cover its syntax, defining joints and links, and basic kinematics.
